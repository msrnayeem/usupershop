## 🔧 STEP 1 — Google Search Console (সবার আগে করুন)

Google Search Console আপনার site Google-কে চেনাবে।

### A. Account খুলুন
1. যান → **https://search.google.com/search-console**
2. Google account দিয়ে login করুন (Gmail)
3. **Start now** বাটনে click করুন

### B. Property Add করুন
1. **Add property** → **URL prefix** select করুন
2. URL দিন: `https://usuper.shop`
3. **Continue** চাপুন

### C. Verify করুন (HTML file method সহজ)
1. **HTML file** option select করুন
2. একটা file download হবে যেমন: `google1234abc.html`
3. এই file টি আপনার server-এ **public/** ফোল্ডারে upload করুন
4. তারপর **Verify** বাটন চাপুন
5. "Ownership verified" দেখালে সফল ✅

### D. Sitemap Submit করুন
1. বাম মেনু → **Sitemaps**
2. **Add a new sitemap** → টাইপ করুন: `sitemap.xml`
3. **Submit** চাপুন
4. Status: **Success** দেখালে সফল ✅

### E. Indexing Request করুন
1. বাম মেনু → **URL Inspection**
2. উপরে URL দিন: `https://usuper.shop`
3. **Request Indexing** বাটন চাপুন
4. একইভাবে এগুলোও করুন:
   - `https://usuper.shop/product-list`
   - `https://usuper.shop/about-us`
   - `https://usuper.shop/seller/signup`
